# 👁️ Computer Vision & Edge AI Portfolio

Цей репозиторій є портфоліо моїх навчальних та практичних проектів у сфері комп'ютерного зору (Computer Vision) та штучного інтелекту. 
Головний фокус — обробка відео в реальному часі, розпізнавання об'єктів та інтеграція AI-алгоритмів з апаратними платформами (Edge AI / IoT).

## 🛠 Технічний стек
* **Нейромережі (Deep Learning):** YOLOv8 (Ultralytics), Object Detection.
* **Обробка зображень:** OpenCV (Python), обробка MJPEG-потоків.
* **Web & Backend:** FastAPI, Uvicorn, Asynchronous Python (для стрімінгу відео без затримок).
* **Мережа та Інтеграція:** HTTP/REST, взаємодія з мікроконтролерами (ESP32/STM32) через локальну мережу.
* **Інструменти:** Python 3.12 (Miniconda), Linux (Ubuntu), VS Code.

## 📂 Огляд проектів

Репозиторій поступово наповнюється. Нижче представлені поточні розробки, розділені за рівнем складності:

### 🔹 Рівень 4: Комплексні AI-системи (End-to-End Vision)
Системи, що охоплюють повний цикл: захоплення відео, нейромережевий інференс та передача керуючих сигналів на виконавчі механізми.

| Проект | Опис | Технології |
| :--- | :--- | :--- |
| **[Edge-AI_PTZ_System](./Edge-AI_PTZ_System)** | **Інтелектуальна камера з віддаленим керуванням.** <br> Сервер на FastAPI захоплює відео з бездротової камери ESP32-CAM, обробляє його через **YOLOv8** для розпізнавання об'єктів (люди, транспорт тощо) і транслює результат у браузер. Реалізовано асинхронний веб-інтерфейс, який дозволяє користувачу керувати сервоприводами камери (на базі STM32) без затримок відеопотоку. | `YOLOv8`, `FastAPI`, `OpenCV`, `ESP32-CAM Stream`, `Asynchronous UI` |

*(Тут з'являтимуться нові проекти: від базових скриптів OpenCV до алгоритмів трекінгу)*

## 📈 Плани на майбутнє (Roadmap)
Моя мета — поглибити практичні навички в Computer Vision та оптимізації продуктивності AI. Що в планах:
* [ ] **Object Tracking:** Реалізація алгоритмів автоматичного слідування камери за рухомим об'єктом.
* [ ] **Face Recognition:** Створення локальної системи розпізнавання облич для контролю доступу.
* [ ] **Model Optimization:** Конвертація моделей YOLO у формати TensorRT або ONNX для підвищення FPS на процесорах та Edge-пристроях.
* [ ] **Depth Estimation:** Робота зі стереозором або оцінка відстані до об'єкта за його розмірами в кадрі (Bounding Box).
